
# Calendar App




| Variable          | Entorno     | Descripción                                          |
| ----------------- | ----------- | ---------------------------------------------------- |
| `VITE_API_URL`    | Ambos       | URL base del backend (API REST)                      |
| `VITE_SW`         | Prod        | Activa el modo Service Worker completo               |
| `VITE_SW_DEV`     | Dev         | Activa el modo SW de desarrollo                      |
| `VITE_CLAIMS`     | Prod        | Permite que el SW reclame clientes automáticamente   |
| `VITE_RELOAD_SW`  | Prod        | Fuerza recarga cuando hay nueva versión del SW       |
| `VITE_SW_DESTROY` | Dev (debug) | Destruye el SW y cache (para limpiar builds previas) |
